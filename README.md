##  Hello! This is my app
# Grocery Shopping List Manager

-----------

## What is this app about?
A mobile app designed to help users manage and organize the items they need to buy for their grocery trips. With a clean interface and a simple workflow, the app streamlines grocery planning and makes it easier for users to stay organized and focused during shopping.
 
### Notable features of the app
1. Add, edit, and delete grocery items
2. Categorize items by type (e.g., Fruits & Vegetables, Meat & Seafood, Dairy & Eggs)
3. Display item previews with image and key details (name, price, quantity, units, notes)
3. View a real-time summary including total price and items
4. Navigate between home list, add form, and edit form seamlessly
5. Save and load item data using AsyncStorage for persistent storage
